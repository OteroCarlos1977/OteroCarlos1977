## Perfil profesional

Creo aplicaciones web simples y funcionales para ordenar procesos reales. Combino desarrollo, datos y experiencia en gestión para convertir turnos, expedientes, catálogos y reportes en herramientas claras, útiles y accesibles.

Trabajo principalmente con React, Vite, JavaScript, Node.js, Express, APIs REST y bases de datos SQL/SQLite/MySQL. También incorporo herramientas de datos como Excel, Google Sheets, Python, Power BI, Looker y Google Colab para transformar información dispersa en reportes y tableros útiles.

Busco integrarme a equipos como desarrollador frontend o full stack junior, especialmente en proyectos de gestión, automatización, dashboards y soluciones internas para organizaciones.
